Instructions:
  1. Install node js
  2. Open your terminal
  3. Run the command 
    npm install -g artillery
  4. Download the test file
  5. Point your terminal to wherever you downloaded the file. If it's in Downloads, do 
    cd Downloads
    If it's on your desktop do 
    cd Desktop
  6. Run the test by running the command 
    artillery run advanced-test.yaml
